# Gal · Landing Pages

פרויקט Vercel סטטי שמארח דפי נחיתה עם חלוקה ל**אישי** ו**עסקי**.

## מבנה התיקייה

```
.
├── vercel.json
├── index.html           # Hub עם סקציות אישי/עסקי
├── README.md
├── personal/            # דפי נחיתה אישיים
│   └── gagler-30/
└── business/            # דפי נחיתה עסקיים (ריק לעת עתה)
```

## URLs

| URL                        | מה זה                          |
|----------------------------|--------------------------------|
| `/`                        | Hub עם סקציות אישי/עסקי        |
| `/personal/gagler-30`      | מסיבת גג 30, 06.06.2026        |
| `/business/<slug>`         | (עתידי) דף עסקי                |

## להוספת דף חדש

1. צור תת-תיקייה ב-`personal/` או `business/` בשם ה-slug.
2. הכנס `index.html` (וכל ה-assets).
3. עדכן את `index.html` בשורש — הוסף `<li>` בסקציה המתאימה.
4. Commit ב-GitHub → Vercel פורס אוטומטית תוך 30 שניות.
